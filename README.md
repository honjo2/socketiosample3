Expressの雛形にsocket.ioでチャット機能を加えたものです

[Node Ninja]用に作りました。
[これ][beta]がそう。

ローカルで動かしたい場合は

>% git clone git@github.com:honjo2/socketiosample3.git socketiosample3

>% cd socketiosample3

>% npm install

>% vi server.js // ポートを適当に修正

>% node server.js

>http://localhost:****/にアクセス

Node Ninjaの使い方は[ここ][getting]を参考に
